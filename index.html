Часто задаваемые вопросы

Как скачать приложение?
Подробная инструкция по установке StreamVibe:

Шаг 1: Скачивание архива
Нажмите кнопку "Скачать для Windows" на главной странице. Вы будете перенаправлены на MediaFire, где доступен архив streamvibe_setup.zip размером всего 12 МБ. Файл полностью безопасен и проверен антивирусами.

Шаг 2: Распаковка архива
Архив защищен паролем для безопасности. Пароль: 3987. Введите его при распаковке. Рекомендуем использовать WinRAR или встроенный архиватор Windows. Распаковка занимает менее 10 секунд.

Шаг 3: Запуск установщика
В распакованной папке найдите файл stream-vibe-installer.exe. Дважды кликните по нему для запуска. Установщик автоматически определит ваше разрешение экрана и настроит оптимальные параметры.

Шаг 4: Процесс установки
Следуйте простым инструкциям на экране. Установка занимает 2-3 минуты. Выберите папку установки (по умолчанию: C:\Program Files\StreamVibe). Установщик создаст ярлык на рабочем столе и в меню "Пуск".

Готово!
После завершения установки перезапустите приложение. При первом запуске создайте аккаунт или войдите как гость. Вы сразу сможете создавать комнаты и смотреть фильмы с друзьями! Техподдержка доступна 24/7 через чат в приложении.

Как синхронизировать видео?
Технология синхронизации StreamVibe:

Автоматическая синхронизация:
При создании комнаты StreamVibe использует нашу проприетарную технологию "SyncTime™", которая автоматически синхронизирует воспроизведение видео для всех участников. Задержка составляет менее 200 мс даже при слабом соединении.

Как это работает:
1. Хост комнаты (создатель) управляет воспроизведением
2. Все остальные участники получают команды синхронизации в реальном времени
3. Видео автоматически ставится на паузу/воспроизводится/перематывается одновременно
4. Встроенная система компенсации задержки автоматически подстраивается под разную скорость интернета

Ручная синхронизация:
Если автоматическая синхронизация не сработала, используйте кнопку "Синхронизировать" в панели управления комнатой. Это мгновенно выровняет все видео по текущему времени хоста.

Поддерживаемые форматы:
YouTube (включая 4K), Netflix, Amazon Prime, Disney+, HBO Max. Для локальных файлов поддерживается формат MP4 с кодеками H.264/H.265.

Советы:
• Используйте стабильное Wi-Fi соединение
• Отключите VPN для лучшей синхронизации
• Убедитесь, что все участники используют одинаковую версию приложения

Можно ли смотреть оффлайн?
Оффлайн возможности StreamVibe:

Онлайн режим (основной):
Для синхронного просмотра с друзьями обязательно требуется интернет-соединение. Минимальная скорость: 5 Мбит/с для 1080p, 25 Мбит/с для 4K. StreamVibe автоматически подстраивается под качество вашего соединения.

Оффлайн скачивание:
Локальные файлы: Загружайте любые MP4/MKV файлы на компьютер и смотрите оффлайн. Поддержка до 8K разрешения и всех популярных аудио/видео кодеков.

Из стриминговых сервисов:
• Netflix - Скачивайте фильмы/сериалы через официальное приложение Netflix на Windows
• YouTube Premium - Оффлайн скачивание видео (требуется подписка)
• Amazon Prime - Скачивание контента для оффлайн просмотра

Локальная синхронизация:
StreamVibe поддерживает "локальную сеть" режим. Создайте комнату в локальной сети Wi-Fi без интернета. Все участники должны быть в одной сети. Синхронизация работает через multicast протокол с задержкой менее 100 мс.

Хранение оффлайн контента:
• Автоматическое создание папки C:\StreamVibe\Offline
• Поддержка внешних накопителей (USB, HDD)
• Встроенный медиаплеер с субтитрами и эквалайзером

Ограничения оффлайн режима:
• Чат работает только в локальной сети
• Нет облачного сохранения прогресса просмотра
• Ограниченная поддержка DRM-защищенного контента

Рекомендации:
Для лучшего оффлайн опыта используйте локальные MP4 файлы или скачивайте контент заранее через официальные приложения сервисов.
